# APSpace Auto Attendance Bot 🤖 (i did it with my brother)

An automated script designed to handle APSpace portal login and automatically submit attendance OTP codes for multiple accounts. 

## 🚀 Features
* **Multi-Account Support:** Easily manage multiple student accounts via a YAML configuration file.
* **Automated Login & Check-in:** Simulates the login process and inputs the OTP code automatically.
* **Convenience-Focused:** Built to streamline and automate repetitive portal navigation.
